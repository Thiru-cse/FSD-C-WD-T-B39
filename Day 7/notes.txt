Margin, Padding, Border

Position Properties:
need: even though if you have top, left, bottom and right,
      which can be used to place/align the element on the page.
      1. Static
      2. Relative
      3. Absolute
      4. Fixed
      5. Sticky

1.Static Example:
<div class="main">Static</div>
.main{
    height: 150px;
    width: 200px;
    background-color: lightblue;
    font-family: 'PT Sans', sans-serif;
    position: static;
    top: 50px;
}

2.Relative Example:
<div class="main">Static</div>
.main{
    height: 150px;
    width: 200px;
    background-color: lightblue;
    font-family: 'PT Sans', sans-serif;
    position: relative;
    top: 50px;
    left: 150px;
}

3. Absolute Example:
<div class="parent">
    <div class="child">Absolute</div>
</div>
.parent {
    height: 300px;
    width: 300px;
    background-color: lightgreen;
    margin: 50px;
    position: relative;
}

.child{
    background-color: salmon;
    position: absolute;
    top: 50px;
    left: 50px;
}
