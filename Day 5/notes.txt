Day 5 - HTML

What is HTML? Why use it!

HTML - Hyper Text Markup language
HTML vs HTML5
Head Tag: Meta tag-information about the document
Body Tag: Contain all of your document.
Element: Tag + Content
Example: <h1>Welcome</h1>
above one is called Element.

Heading Tag: h1...h6

Paragraph Tag:<p>Some Paragraph</p>

Break Tag: <br> - line break

A Tag:
To redirect the document, we use "a tag/anchor tag".
Example:
<a href="https://github.com/Thiru-cse/FSD-C-WD-T-B39" target="_blank">Session Notes</a>
attribute: it gives the additional information about the tag.
href: represent the link where it has to be redirected.
target: help us to open the document on the new tab.

Semantic Tags:
Header, Navigation, Main, Section, Aside, Article, Footer

Example:
<header>
    <h1>Website Header</h1>
    <nav>
        <a href="#home">Home</a>
        <a href="#about">About</a>
        <a href="#Profile">Profile</a>
    </nav>
</header>
<main>
    <article>
        <h2>Article Title</h2>
        <p>This is an article paragraph</p>
    </article>
</main>

<aside>
    <p>Related links or advertisement go here</p>
</aside>

<footer>
    <p>Website footer information here.</p>
</footer>
